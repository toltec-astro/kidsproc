OpticalDetune¶

class kidsproc.kidsmodel.OpticalDetune(*args, **kwargs)[source]¶

Bases: kidsproc.kidsmodel._Model

Model that describes detuning of KIDs in response to incident optical power.

The model reads

x = (p - background) * responsivity
